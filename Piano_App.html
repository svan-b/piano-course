Make it a real home screen app

Hosted (best): put this whole folder on any free static host (GitHub Pages: new repository, upload files, Settings, Pages, deploy from main). Open the URL in Safari on the iPad, tap Share, then Add to Home Screen. You get a fullscreen app with an icon, offline support, and reliable saved progress.

Local (works now): open this file from the Files app. Progress saves in this viewer, but iOS treats local files more casually than websites, so use Export backup weekly.
